# Ford Sprint HTML/CSS

# Objetivo

  Projeto realizado durante a sprint de HTML e CSS do curso Ford Enter, com o objetivo
  de implementar as habilidades adquiridas na sprint construindo uma aplicação web.

# Funcionalidades 
 
  O projeto consta com três paginas e com focos diferentes:

  - Home
  - Lançamento
  - Contato
